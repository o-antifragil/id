O que são os ID's?

No CSS, o elemento id é um seletor usado para estilizar um elemento HTML específico e único na página. Ao contrário de classes (que podem ser aplicadas a múltiplos elementos), o id deve ser único por página e só pode ser aplicado a um único elemento.

Características do id:

  1. Símbolo #: No CSS, o id é selecionado com o símbolo #, seguido pelo nome do id.
  2. Especificidade Alta: O id possui uma especificidade maior que as classes, portanto, regras de id sobrescrevem regras de classe se aplicadas ao mesmo elemento.
  3. Identificação Única: É usado quando você deseja aplicar estilos ou ações JavaScript em um elemento específico e único.

Exemplo de Uso

No HTML:

            
<div id="principal">
    Conteúdo Principal
</div>

No CSS:

            
#principal{
    background-color: lightblue;
    padding: 20px;
}

Boas Práticas

  • Use o id somente quando realmente precisar estilizar um único elemento.
  • Prefira classes (class) para estilos que possam ser aplicados a vários elementos.

O uso de id ajuda a identificar e estilizar elementos únicos na página com precisão e especificidade.
